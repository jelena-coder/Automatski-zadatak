# Automatski-zadatak
Automatski zadatak, Sitemap page, Destination page

## Pregled

Sadrzi UI testove za stranicu koristeci Selenium, JUnit 5, Maven, Java21 and WebDriverManager.

## Tech Stack
- Java 21
- Selenium WebDriver (4.18.1)
- JUnit 5
- WebDriverManager (5.7.0)
- Maven

## Test Scenarios
- Navigacija: Sitemap do Destination page
- Validacija cartera
- Validacija funkcionalnosti sortiranja (Lowest and Highest Price)
